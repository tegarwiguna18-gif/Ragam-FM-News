Cara Menulis & Menambahkan Artikel Sendiri

  1. Buat file baru dengan format: 2026-07-10-judul-artikel-saya.html
  2. Salin kerangka artikel di bawah, ganti judul, tanggal, dan isi dengan tulisanmu
  3. Unggah file ke repositori ini
  4. Di halaman utama, tambahkan kartu baru dengan link menuju file tersebut

Sangat mudah! Tanpa butuh aplikasi tambahan, cukup tulis langsung di GitHub.
